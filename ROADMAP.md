# 🗺️ AI Engineer Roadmap — From Zero to Job Ready

## Why This Roadmap?
Most AI/ML roadmaps are either too vague or too advanced.
This one is built by a beginner, for beginners —
based on real daily learning and job market research.

## Phase 1 — Python Foundations (4 weeks)
### Week 1
- Variables, Data Types, I/O
- Conditions and Loops
- Functions

### Week 2
- Data Structures (Lists, Tuples, Dicts, Sets)
- File Handling
- Exception Handling

### Week 3
- OOP — Classes, Inheritance
- OOP — Encapsulation, Polymorphism
- Abstract Classes

### Week 4
- Modules and Packages
- Regular Expressions
- Working with APIs
- 🏗️ Project: Student Report Card Generator

## Phase 2 — Math & DSA (4 weeks)
### Week 5
- Arrays, Searching, Sorting
- Stacks and Queues
- Linked Lists

### Week 6
- Linear Algebra — Vectors, Matrices
- Statistics — Mean, Median, Std Dev
- Probability Basics

### Week 7
- NumPy — Arrays and Operations
- Pandas — DataFrames and Cleaning
- Matplotlib — Data Visualization

### Week 8
- Exploratory Data Analysis
- Feature Engineering
- 🏗️ Project: Data Analysis Dashboard

## Phase 3 — Machine Learning (6 weeks)
### Week 9-10
- ML Types and Workflow
- Linear and Logistic Regression
- Decision Trees and Random Forest

### Week 11-12
- SVM, KNN, Naive Bayes
- Model Evaluation and Metrics
- Cross Validation

### Week 13-14
- NLP Basics
- Sentiment Analysis
- 🏗️ Project: House Price Predictor
- 🏗️ Project: Spam Classifier

## Phase 4 — Deep Learning & AI (6 weeks)
### Week 15-16
- Neural Networks from scratch
- TensorFlow and Keras
- CNN for Image Classification

### Week 17-18
- RNN and LSTM
- Transformers
- HuggingFace

### Week 19-20
- LLMs and Prompt Engineering
- LangChain and AI Agents
- Model Deployment
- 🏗️ Final Portfolio Project
