## 🔑 Core Concepts

| Concept | Meaning                             |
| ------- | ----------------------------------- |
| Module  | A single Python file                |
| Package | A folder containing modules         |
| Library | Collection of packages              |
| import  | Loads code from another module      |
| venv    | Creates isolated Python environment |
| pip     | Installs third-party libraries      |

## 💡 Key Lessons

* Modules help split code into reusable files.
* Packages organize multiple modules.
* Virtual environments prevent dependency conflicts.
* Most AI libraries are large package collections.
* Imports improve code organization.

---

## 🤖 AI/ML Connection

Scikit-learn structure:

sklearn/
├── linear_model.py
├── tree.py
├── ensemble.py
└── metrics.py

When we write:

from sklearn.linear_model import LinearRegression

we are importing from a package exactly like our custom package.
