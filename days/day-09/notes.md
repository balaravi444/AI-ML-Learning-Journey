## 💡 Key Lessons

- try block runs the code that might cause an error
- except block handles the error gracefully
- else block runs only when no error occurs
- finally block always runs no matter what
- Custom exceptions make error messages more meaningful

## 💎 Important Realization

Good programming is not just about writing code that works — it's about writing code that can handle unexpected situations gracefully.
