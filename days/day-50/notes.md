## 🔑 Deployment Steps

```bash
# 1. Ensure requirements.txt is complete
cat requirements.txt

# 2. Add render.yaml to project root
# (already created — see below)

# 3. Push to GitHub
git add .
git commit -m "Indian Job Market Analyzer v1.0 - Deploy"
git push

# 4. Go to render.com
# New → Web Service → Connect GitHub
# Root Dir: projects/indian_job_market_analyzer
# Build: pip install -r requirements.txt
# Start: uvicorn app:app --host 0.0.0.0 --port $PORT

# 5. Wait 3-5 mins → Live URL! 🎉
```
